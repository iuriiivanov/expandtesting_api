## Условия для запуска тестов

Проект находится в стадии доработки.
На данном этапе для запуска тестов необходимо в корне проекта создать файл .env с содержимым вида API_DEFAULT_TOKEN="TOKEN". TOKEN можно получить, создав нового пользователя при помощи Swagger.
